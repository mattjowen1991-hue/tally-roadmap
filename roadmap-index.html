Phase 2 — UX Polish
Micro-interactions & Feel
Small touches that make the app feel premium. Animations, haptics, transitions, and visual refinements.
UI Polish
Panel transition animations
Smooth slide transitions between panels with spring easing, rather than instant swaps
Card entry animations
Staggered fade-in when switching panels. Bills, debts, savings cards animate in sequentially.
Haptic feedback (Capacitor)
Light vibration on button taps, toggle paid, swipe panel change. Uses Capacitor Haptics plugin.
Toast notifications
"Bill added", "Payment recorded", "Goal reached!" — subtle slide-up toasts instead of silent changes
Swipe-to-delete on bill cards
Swipe left to reveal delete button on individual bills, debts, savings — like iOS Mail
Pull-to-refresh feel
Pull down on any panel to trigger a visual refresh with the Tally logo as the spinner
Phase 3 — Features
Monthly Reports & Insights
Give users real insight into their spending habits. End-of-month summaries, trends over time, and smart nudges.
Feature Logic
Monthly snapshot storage
On month rollover, save a snapshot: total income, expenses per category, bills paid/missed, debt reduced, savings deposited
Monthly summary view
New panel or modal: "February Summary" — income vs expenses, biggest categories, missed bills, debt progress
Spending trends chart
Line chart showing monthly expenses over time. Uses stored snapshots. Chart.js line/bar integration.
Bill search
Search/filter input on the Bills panel to find specific bills by name across all categories
Sort bills
Sort by: due date, amount (high/low), name, status. Dropdown on the Bills panel.
Phase 4 — Multi-Platform
Desktop Layout & iOS
Expand beyond mobile. A proper desktop layout with sidebar navigation, and an iOS build for the App Store.
UI Feature
Desktop sidebar navigation
Replace swipe panels with a fixed sidebar on screens > 768px. Icons + labels. Content area to the right.
Desktop bill table view
Sortable table layout for bills on desktop — name, category, projected, actual, status, due date. Inline editing.
Responsive breakpoints
Clean transition between mobile card layout and desktop table layout. Intermediate tablet layout.
iOS build setup
Capacitor iOS config, Xcode project, safe area insets, status bar styling. Test on simulator.
App icons & store assets
Generate app icons from Tally logo. Android adaptive icons. iOS icon set. Store screenshots.
Phase 5 — Data
Export, Backup & Settings
Users need control over their data. Export to CSV, backup/restore, and a proper settings screen.
Feature Logic
Export bills to CSV
Generate and download a CSV of all bills, debts, savings with dates and amounts
Backup & restore
Export all data as JSON file. Import from JSON to restore. Useful for switching devices.
Settings panel
Currency symbol, theme (auto/dark/light), reset all data, about screen, version info
Notification reminders
Capacitor Local Notifications — remind user of upcoming bills, unpaid items, savings goals
